# customer-spending-analysis
Customer spending behavior analysis and predictive modeling using MEPS dataset
## Project Overview
This project analyzes customer spending behavior using simulated MEPS dataset to identify high-cost individuals and predict healthcare expenditure.

## Key Techniques
- Data Cleaning & Feature Engineering
- Classification (Random Forest)
- Regression (Linear Regression)
- Data Visualization (Seaborn, Matplotlib)

## Results
- Classification Accuracy: ~47%
- AUC Score: ~0.47
- R² Score: ~0.004

## Business Impact
- Identify high-risk customers
- Optimize resource allocation
- Improve decision-making using data

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
